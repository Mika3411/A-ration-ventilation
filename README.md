# Aération Ventilation

Site e-commerce et vitrine pour Aération Ventilation.

Stack principale :

- React + Vite pour le front.
- Node.js + Express pour les API et le rendu SEO.
- PostgreSQL pour les comptes client, produits admin, catégories, codes promo et commandes.
- Stripe Checkout pour le paiement.
- Nodemailer/SMTP pour le formulaire de contact.
- Render pour l'hébergement applicatif recommandé.
